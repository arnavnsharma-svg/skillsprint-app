# SkillSprint 🚀

SkillSprint is a developer-focused micro-gig platform that replaces traditional resumes with a SkillScore system.

## Features
- SkillScore Dashboard
- Micro-Gig Marketplace
- 7-Day Sprint System
- Escrow Wallet
- Proof-of-Work Portfolio

## Tech Stack
- Flutter
- Dart
